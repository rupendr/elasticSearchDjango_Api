# elasticSearchDjango


## Installation

Install packages.

```bash
pip install -r requirements.txt
```
```bash
python manage.py makemigrations
```
```bash
python manage.py migrate
```
```bash
python manage.py search_index --rebuild
```
```bash
python manage.py runserver
```
